# 坦克大战 - Battle City

经典坦克大战 HTML 游戏，使用纯 JavaScript + Canvas 实现。

## 功能特性

- 玩家坦克控制（WASD 移动，空格开火）
- 敌方坦克 AI，三种类型（普通、重型）
- 多关卡，不同地图布局
- 可破坏的砖墙、不可摧毁的钢墙
- 基地保护玩法
- 爆炸特效
- 得分系统

## 操作说明

| 按键 | 操作 |
|------|------|
| W/A/S/D 或 方向键 | 移动坦克 |
| 空格 | 开火 |
| P | 暂停 |

## 玩法

- 消灭所有敌方坦克即可进入下一关
- 保护底部中央的基地（黄色星星）
- 基地被摧毁或生命耗尽则游戏结束
- 敌方坦克从顶部三个位置随机生成

## 开始游戏

直接在浏览器中打开 `index.html` 即可游玩。
